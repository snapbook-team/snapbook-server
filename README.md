# Description

Backend service for the [Snapbook](https://github.com/snapbook-team)

# Bootstrap

## Dependency Installation

```
npm i -g yarn
yarn install
```

## Build

```
yarn build
```

# Start

## Production mode

```
yarn start:prod
```

## Development mode

```
yarn start:dev
```
