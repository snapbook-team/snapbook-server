# Description

Backend service for the [Snapbook](https://github.com/snapbook-team)

# Bootstrap

## Dependency Installation

```
npm i -g pnpm
pnpm install
```

## Build

```
pnpm run build
```

# Start

## Production mode

```
pnpm run start:prod
```

## Development mode

```
pnpm run start:dev
```
