# Description

Template of a gateway server for managing gRPC modules.
This template is based on the use of submodules

# Bootstrap

## Dependency Installation

```
npm i -g yarn
yarn install
```

## Build

```
yarn build
```

# Start

## Production mode

```
yarn start:prod
```

## Development mode

```
yarn start:dev
```
